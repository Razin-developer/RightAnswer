# RightAnswer Web

This is the new React/Vite web app. The old Next.js app is preserved in
`apps/web-next-legacy` until every page has parity.

## Pages

- Home landing page
- App Features
- Documentation
- Admin metrics

## Environment

```bash
VITE_API_URL=http://localhost:4000/api
```

## Commands

```bash
pnpm --filter @right-answer/web dev
pnpm --filter @right-answer/web build
```
